# Financial Church

Sistema de gestão financeira para igrejas utilizando Google Apps Script e Google Sheets.

## Tecnologias

- Google Apps Script
- Google Sheets
- HTML Service
- JavaScript (V8)

## Estrutura

- Dashboard
- Cadastro de membros
- Movimentações
- Contas a pagar
- Relatórios
- Controle de dizimistas

## Desenvolvimento

Instalar dependências

npm install

Enviar alterações

npm run push

Abrir projeto

npm run open
